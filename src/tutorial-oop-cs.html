1. Lépés

Mi az az OOP?

Az Objektumorientált Programozás (OOP) egy szemléletmód, ahol a programot egymással kommunikáló objektumok halmazaként fogjuk fel. Gondolj egy objektumra úgy, mint egy valós világban létező dologra: egy autó, egy felhasználó, vagy akár egy bankszámla.

Minden objektumnak vannak:

  • Tulajdonságai (Adatai): pl. az autó színe, a felhasználó neve.
  • Képességei (Metódusai): pl. az autó elindul, a felhasználó bejelentkezik.
2. Lépés

Osztály vs. Objektum

Az Osztály a tervrajz, az Objektum pedig a tervrajz alapján elkészült konkrét dolog.

Nézzünk egy egyszerű C# példát egy kutya osztályra:

public class Kutya 
{
    // Tulajdonságok
    public string Nev;
    public int Kor;

    // Metódus (Képesség)
    public void Ugat() 
    {
        Console.WriteLine(Nev + " mondja: Vau-vau!");
    }
}

És így hozzuk létre belőle az objektumot:

Kutya fido = new Kutya();
fido.Nev = "Fidó";
fido.Kor = 3;
fido.Ugat(); // Kimenet: Fidó mondja: Vau-vau!
3. Lépés

Próbáld ki!

A fenti kódban a new kulcsszóval hoztunk létre egy új példányt (objektumot) az osztályból. Próbáld meg elképzelni, hogyan nézne ki egy Auto osztály!

Feladat: Tervezz meg fejben egy Autó osztályt 3 tulajdonsággal és 1 metódussal!
